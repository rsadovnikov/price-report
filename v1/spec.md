# v1: Отчёт о цене и конкурентах — первый прототип

> Файлы: `index.html` → `report.html`

---

## Описание

Первая версия прототипа страницы «Отчёт о цене и конкурентах» для ProApp.

Основные блоки:
- Таблица A — объект оценки (верхняя, фиксированная)
- Таблица B — таблица конкурентов с горизонтальным скролом
- Степпер справа с разделами: Обзор конкурентов, Рекомендованная цена, Статистика, Комментарий агента
- Блок «Отчёт для собственника» как accordion под основным контентом
- Sticky footer с кнопками «Создать PDF» и «Сохранить и выйти»

## Ключевые особенности

- Две вкладки: «В отчёте» и «Подборка конкурентов» (активна по умолчанию — «Подборка»)
- Комментарии к конкурентам добавляются прямо в таблице
- Sticky footer всегда виден при прокрутке
