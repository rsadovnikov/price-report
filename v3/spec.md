## Что изменилось относительно v2

### 1. Вкладки и навигация

| v2 | v3 |
|----|----|
| «Подборка конкурентов» (активна по умолчанию) | «В отчёте» (активна по умолчанию) |
| «В отчёте» | «Конкуренты N» |

### 2. Вкладка «В отчёте» — пустое состояние

- Показывается, когда не выбран ни один конкурент.
- Одна большая кнопка «Выбрать конкурентов» вместо прежних двух маленьких.
- Клик на кнопку → переключает на вкладку «Конкуренты».

### 3. Блок «Создать отчёт для собственника»

Оформлен как отдельная карточка ниже основного отчёта.

**Заблокирован**, пока не выбран ни один конкурент — вместо содержимого показывается заглушка.

**Разблокируется**, как только выбран хотя бы один конкурент.

#### Внутренний степпер (4 шага)
1. **Выбраны N конкурентов** — раскрываемый список выбранных конкурентов с возможностью добавить комментарий к каждому; по умолчанию свёрнут.
2. Рекомендованная цена
3. Статистика
4. Комментарий агента

#### Комментарии к конкурентам

Кнопка «Добавить комментарий» перенесена из основной таблицы в список внутри шага 1. Там же — просмотр, редактирование и удаление существующего комментария.

### 4. Sticky footer убран

Кнопки «Создать PDF-отчёт» и «Сохранить и выйти» перемещены в нижнюю часть блока «Создать отчёт для собственника».

## Ключевые состояния

### Начальный экран (нет выбранных конкурентов)
- Активна вкладка «В отчёте».
- Показывается empty-state с кнопкой «Выбрать конкурентов».
- Блок «Создать отчёт для собственника» заблокирован, виден placeholder.

### После выбора ≥1 конкурента
- Вкладка «В отчёте» показывает таблицу выбранных.
- Блок «Создать отчёт для собственника» разблокируется.
- Шаг 1 отображает «Выбраны N конкурентов» (N изменяется динамически).
- Список конкурентов в шаге 1 по умолчанию свёрнут.

### Комментарий к конкуренту
- «Добавить комментарий» появляется только в списке внутри шага 1.
- Inline-форма открывается прямо в строке; Enter сохраняет, Escape/«Отмена» закрывает.
- При наведении на пузырь — появляются кнопки редактирования и удаления.
